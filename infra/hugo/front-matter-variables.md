# Front matter variables

**Front matter** allows you to keep metadata attached to an instance&#x20;

```yaml
+++
slug = 'test'
url = '/blog/test'
title = 'My First Post'
description = 'Pots description goes here.'
tags = ['personal', 'test']
categories = ['Development', 'VIM']
date = '2012-04-06'
draft = 'false'
+++
```
